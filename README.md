# cat-backend
[관광데이터 공모전 출전작] CaT(Camping Trade)의 웹 서버입니다.🏕
## 📁 프로젝트 소개
캠핑장 예약 및 캠핑 물품 나눔 플랫폼 CaT(Camping Trade)의 웹 서버입니다.
<br/>

## 👩🏻‍💻 개발 기간
* 2022.06.18 ~ 2022.09.15

## 🗳 개발 환경
- `Java 11`
- **IDE**: IntelliJ
- **Framework**: Spring Boot (2.7.1)
- **Database**: AWS RDS (mariaDB)
